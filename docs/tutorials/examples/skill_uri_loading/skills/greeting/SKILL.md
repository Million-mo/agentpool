# Skill: greeting

Generate personalized greetings with customizable tone and format.

## License

MIT

## Compatibility

1.0.0

## Allowed Tools

All

## Instructions

Create a greeting for $1 from $2.
Use a $3 tone.

The greeting should:
1. Address $1 by name
2. Mention they are from $2
3. Match the $3 tone (formal, casual, friendly, professional)
4. Be concise but warm

Example outputs:
- Formal: "Dear Alice, welcome from Company. We appreciate your presence."
- Casual: "Hey Alice! Great to have you here from Company!"
- Friendly: "Hi Alice! So glad you're here with us from Company!"
