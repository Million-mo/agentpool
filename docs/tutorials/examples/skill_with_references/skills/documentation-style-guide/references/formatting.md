# Formatting Guidelines

## Code Blocks

Use fenced code blocks with language identifiers:

```python
# Good: Language specified
print("Hello, world!")
```

Inline code for: `variable_names`, `function_calls()`, `ClassNames`

## Text Formatting

- **Bold** for UI elements and emphasis
- *Italics* for new terms and definitions
- `Code` for technical terms

## Lists

Use bullet points for unordered items:
- First item
- Second item
  - Nested item

Use numbered lists for sequential steps:
1. First step
2. Second step
3. Third step

## Links

- Internal: [Section Title](#section-id)
- External: [External Site](https://example.com)
- Cross-doc: [Other Doc](../other/page.md)

## Callouts

Use admonitions for important notes:

!!! note
    Supplementary information

!!! warning
    Critical warnings

!!! tip
    Helpful suggestions
